### Node.js 简介

- Node.js 是一个能够在服务器端运行 JavaScript 的跨平台 JavaScript 运行环境
- Node 采用 Google 开发的 V8 引擎运行 js 代码，使用**事件驱动**、**非阻塞**和**异步 I/O 模型**等技术来提高性能，可优化应用程序的传输量和规模

### Hello World

编写 hello.js 文件：

```javascript
console.log("Hello World");
```

在命令行中输入：`node hello.js` 即可运行了。
